# Análisis Bayesiano de víctimas de delitos sexuales en menores de edad en Bogotá D. C. (2012–2022)

Estudio de caso desarrollado para el curso de Estadística Bayesiana de la Universidad Nacional de Colombia.

## Descripción

Este proyecto analiza el conteo de víctimas de delitos sexuales en menores de edad en Bogotá D. C. durante el periodo 2012–2022, utilizando modelos Gamma-Poisson bajo los paradigmas Bayesiano y frecuentista.

El objetivo principal es comparar las tasas promedio de victimización entre hombres y mujeres y evaluar si existen diferencias estadísticamente significativas.

## Metodología

- Modelos Gamma-Poisson
- Inferencia Bayesiana
- Bootstrap paramétrico
- Métodos Monte Carlo
- Intervalos de credibilidad y confianza
- Simulación de tasas de cobertura
- Análisis de sensibilidad
